sony-gpsassist-update
=====================

Automatically Update GPS Assist data on Linux

= Installation
$ ./install.sh

This will
* copy the script to /usr/bin/
* copy the .cfg to /etc/
* copy the udev rules.d file to /etc/udev/rules.d/

= Usage
Edit SONY_FS_UUIDS in /etc/sony-gpsassist-update.cfg
